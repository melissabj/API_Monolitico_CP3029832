# API_Monolitico_CP3029832
Atividade de API simples para conexão com banco de dados utilizando Node.js com Express, sqlite3 e thunder client.


#**Passos para Execução**

## 1. Criar pasta do projeto
```bash
mkdir api-usuarios
cd api-usuarios
```

## 2. Inicializar o projeto

```bash
npm init -y
```

## 3. Instalar dependências

```bash
npm install express sqlite3
```

ou apenas utilizar:

```bash
npm install
```
   
## 4. Executar aplicação

```bash
node src/app.js
```

A API deverá rodar em:
http://localhost:3000
